# Active Risks (Memory)

_Quick-reference version. Full risk register is in `risks.md`._

## Watch closely

- **Content rot (R1):** IRCC changes policies frequently. Every page needs a "last verified" date in the source log.
- **PR Pathways (R10 — higher risk):** pr-pathways.html covers Express Entry, CEC, PNP, and family sponsorship at a high level. This is the highest-risk page on the site — any phrase that implies eligibility, predicts draws, or recommends a pathway could be read as immigration advice. Language has been kept strictly navigational (may, might, depends on your situation, verify with IRCC). Re-verify all pathway descriptions against IRCC whenever major policy changes are announced (e.g., Express Entry rule changes, new PNP streams, family sponsorship rule updates).
- **Spouse work permit (R9 — higher risk):** IRCC family open work permit rules changed January 21, 2025 and may change again. spouse-work-permit.html is a process-navigation page only — no eligibility claims made. Must be re-verified against IRCC before each deployment cycle. If IRCC changes eligibility criteria, the page may create false impressions even with careful language. Review for currency whenever IRCC announces changes to temporary resident family permits.
- **Scope creep (R8):** Features get proposed constantly. Default is "not yet." See rejected_approaches.md.

## Currently accepted (not mitigated)

- SEO invisibility for 6–12 months — plan around this, not through it
- Single maintainer bottleneck — acceptable at this scale

## Tripwires (stop and review if these happen)

- Any page is linked from a high-traffic source before it's been fully source-verified
- A reader reports acting on information that turned out to be wrong
- A monetization option is proposed that involves editorial content
